  • Element Selectors applies style to all selected elements
  • Class Selectors use period to choose classes, all one-of-us will have a different font
  • ID selectors chooses each ID by using hashtag to choose IDs
selectors can also be grouped, so this text and the id selector will both be font size 20

Combining Selectors

Combining Selectors:
placing selectors next to each other like this:
 div.one-of-us
means every div with the class one-of-us will be selected
this div will be color dark green

Child Selector:
using the larger than sign > means any immediate child of an element that is given is selected
In this case, all spans inside a div will be selected:
 div>span
So this text will turn blue, but the others won't

Descendant Selector:
putting a space between two elements, classes, or IDs will choose all items that is in the first item:
 div span
that will choose all spans that are inside divs
